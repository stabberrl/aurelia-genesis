# Third-party data notices

The optional full lexicons are generated locally from Wiktionary data extracted by [Wiktextract](https://github.com/tatuylonen/wiktextract) and distributed through [Kaikki](https://kaikki.org/). Wiktionary-derived data remains available under CC BY-SA and GFDL terms; it is not relicensed by Aurelia Genesis' Apache-2.0 code license.

Supported editions: Spanish, English, Japanese, Russian, Italian, and French. Generated databases and downloaded dumps are excluded from Git. Each import records its edition, source URL, language, import time, and license notice in SQLite metadata.

When redistributing a generated lexicon, include the relevant Wiktionary attribution, license terms, edition, source URL, dump date, and a notice describing any modifications.

The controlled learning chamber also reads individual pages from the relevant Wiktionary edition through the MediaWiki API. It records the page URL and source name beside every local observation. These observations remain in excluded local SQLite databases and retain Wiktionary's CC BY-SA/GFDL terms; they are not committed or relicensed as project code.
