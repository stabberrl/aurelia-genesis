## Что такое Aurelia Genesis?

Исследовательский проект искусственной жизни, вдохновлённый Fluctlight из *Sword Art Online: Alicization*. Он изучает синтетических когнитивных агентов, которые начинают с минимальными возможностями и постепенно формируют память, связи, предпочтения и операциональный язык из опыта.

LLM не используется как источник интеллекта во время выполнения. AERA служит когнитивной основой, а Genesis управляет изолированными личностями и воспроизводимым рождением. Первая популяция: **Naia, Orin и Iria**.

## Текущие возможности

- Нативный мост AERA и нейтральный протокол событий.
- Эпизодическая память, пластичные связи, забывание, консолидация и распознавание.
- Контролируемое обучение `я/ты/да/нет` с публичным воспроизводимым доказательством.
- Шесть изолированных когнитивных словарей: испанский, английский, японский, русский, итальянский и французский.
- Когнитивный ритм без LLM, который консолидирует опыт и только предлагает внутренние действия.
- Ограниченная по частоте и проверяемая учебная камера Wiktionary, отделяющая внешние наблюдения от доказанного понимания.
- Экспериментальные стадии: зарождение, ранняя, средняя, продвинутая и высокая.
- Раздельные настройки языка интерфейса, когнитивного языка и органического движения.

Стадии развития — экспериментальная оценка опыта, заземления, языка, сенсорного разнообразия и автономных циклов. Они не измеряют сознание, общий интеллект или моральную ценность и будут пересматриваться по мере развития исследования.

## Локальный запуск

```bash
git submodule update --init --recursive
npm install
npm test
npm start
```

Откройте `http://127.0.0.1:4747`. Полные словари импортируются локально по желанию. Проект не утверждает, что создал сознание, биологическую душу или цифровую личность.
