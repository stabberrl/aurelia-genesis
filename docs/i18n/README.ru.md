Публичная версия демонстрирует интерфейс и органическую визуализацию. Для AERA, локального словаря и постоянной когнитивной памяти требуется локальный запуск.

Экспериментальная среда для разработки устойчивых синтетических когнитивных агентов, обучающихся на опыте с помощью AERA.

> Aurelia Genesis исследует когнитивные архитектуры и накопительное обучение. Проект не утверждает, что создаёт сознание, биологические души или цифровых личностей.

## Возможности

- Три воспроизводимые и изолированные идентичности: Naia, Orin и Iria.
- Нативный мост AERA без LLM в рабочем контуре.
- Локальный испанский словарь, постепенное знакомство со словами и сенсорное связывание.
- Базовый курс Naia: 400 понятий и 2 000 семантических связей.
- Научный интерфейс и наблюдаемая карта когнитивного развития.
- Интерфейс на испанском, английском, японском, русском, итальянском и французском.

Когнитивный словарь душ пока доступен только на испанском. Перевод интерфейса не означает многоязычного мышления.

```bash
git submodule update --init --recursive
npm install
npm test
npm start
```

Проект можно изучать, изменять и расширять по лицензии Apache 2.0. Сообщайте о результатах через GitHub Issues, Discussions или pull requests.
