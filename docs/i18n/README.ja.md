## Aurelia Genesisとは？

『ソードアート・オンライン アリシゼーション』のフラクトライトに着想を得た人工生命研究プロジェクトです。最小限の能力から始まり、経験を通じて記憶、関連、好み、操作的な言語を徐々に形成する合成認知エージェントを研究します。

実行時の知能源としてLLMを使用しません。AERAが認知基盤を担い、Genesisが隔離されたアイデンティティと再現可能な誕生を管理します。最初の個体群は **Naia、Orin、Iria** です。

## 現在の機能

- AERAネイティブブリッジと中立イベントプロトコル。
- エピソード記憶、可塑的関連、忘却、統合、認識。
- `私／あなた／はい／いいえ` の管理された学習と公開再現証拠。
- スペイン語、英語、日本語、ロシア語、イタリア語、フランス語の分離された認知辞書。
- LLMを使わず、経験を統合して内部行動のみを提案する認知ハートビート。
- 「発生・初期・中期・高度・高次」の実験的発達段階。
- 表示言語、認知言語、有機的モーションを個別に保存する設定。

発達段階は、経験、感覚との結び付き、言語、感覚の多様性、自律ハートビートから算出する実験的推定です。意識、汎用知能、道徳的価値を測るものではなく、研究の進展に応じて継続的に改訂されます。

## ローカル実行

```bash
git submodule update --init --recursive
npm install
npm test
npm start
```

`http://127.0.0.1:4747` を開いてください。完全な辞書は任意でローカルにインポートできます。プロジェクトは意識、生物学的な魂、デジタル人格を作成したとは主張しません。
