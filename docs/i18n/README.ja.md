# Aurelia Genesis

[Español](../../README.md) · [English](README.en.md) · **日本語** · [Русский](README.ru.md) · [Italiano](README.it.md) · [Français](README.fr.md)

AERAを利用し、経験に基づいて成長する持続的な合成認知エージェントを研究するための実験環境です。

> Aurelia Genesisは認知アーキテクチャと累積学習を研究します。意識、生物学的な魂、デジタル人格を作成したとは主張しません。

## 現在の機能

- 再現可能で互いに隔離された3つのアイデンティティ：Naia、Orin、Iria。
- 実行経路にLLMを使用しないAERAネイティブブリッジ。
- ローカルのスペイン語辞書、段階的な語彙経験、感覚との関連付け。
- Naia向けの400概念と2,000の意味関係からなる基礎カリキュラム。
- 科学的なライブ画面と認知発達マップ。
- スペイン語、英語、日本語、ロシア語、イタリア語、フランス語の画面表示。

魂の認知辞書は現在スペイン語のみです。画面の翻訳は多言語認知を意味しません。

```bash
git submodule update --init --recursive
npm install
npm test
npm start
```
