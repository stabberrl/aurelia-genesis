# Currículo básico de existencia y habla

Este currículo proporciona a Naia 400 conceptos léxicos y 2.000 relaciones semánticas iniciales. Es conocimiento **inyectado**, no experiencia, recuerdo ni creencia adquirida.

Los conceptos cubren veinte dominios: existencia, identidad, lenguaje, pensamiento, percepción, cuerpo, acción, necesidades, emoción, relaciones, sociedad, tiempo, espacio, naturaleza, entorno, cualidades, cantidad, lógica, valores y comunicación.

Las relaciones se construyen de forma reproducible usando menciones en definiciones, sinónimos, antónimos, pertenencia temática y enlaces transversales pendientes de confirmación empírica. Las asociaciones sensoriales continúan almacenándose aparte.

```powershell
npm run curriculum:basic
```

El comando reemplaza únicamente el currículo inyectado del alma indicada. Puede elegirse otra con `node scripts/seed-basic-curriculum.mjs --soul=<id>`.
